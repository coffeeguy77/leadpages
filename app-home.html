Application name

LeadPages by Web Culture

Operated by Bean Culture Pty Ltd trading as Web Culture · ABN 33 600 754 676

The purpose of LeadPages by Web Culture is to help Australian small businesses and agency partners design, publish, and manage websites and landing pages, capture leads, and connect marketing integrations such as Google Ads.

Purpose of LeadPages by Web Culture

LeadPages by Web Culture is a website and landing-page application. Users sign in to the App Command Centre to:

  • Create and publish customer websites and landing pages
  • Edit content, themes, forms, menus, and SEO settings without code
  • Capture and manage leads from published sites
  • Connect Google Ads to measure campaign performance and website conversions

This page is the public application homepage for LeadPages by Web Culture at https://app.leadpages.com.au.

How LeadPages by Web Culture uses Google user data

When a user connects Google Ads, LeadPages by Web Culture uses Google account access to link Google Ads accounts, sync campaign and account performance into the site workspace, and upload website conversion events that help measure lead outcomes.

LeadPages by Web Culture does not sell Google user data and does not share it with unrelated third parties. Details are in the Privacy Policy linked below (same URL as the OAuth consent screen).
