Application name

LeadPages by Web Culture

Operated by Bean Culture Pty Ltd trading as Web Culture. ABN 33 600 754 676.

Purpose of this application

LeadPages by Web Culture is a website and landing-page application for Australian small businesses and agency partners.

The purpose of LeadPages by Web Culture is to let users:

  • Design, publish, and manage websites and landing pages
  • Edit content, themes, forms, menus, and SEO settings without writing code
  • Capture and manage leads from published sites
  • Connect Google Ads to sync campaign performance and upload website conversion events

How LeadPages by Web Culture uses Google user data

When a user connects Google Ads, LeadPages by Web Culture uses Google account access to link Google Ads accounts, sync campaign and account performance into the workspace, and upload website conversion events that help measure lead outcomes.

LeadPages by Web Culture does not sell Google user data and does not share it with unrelated third parties.
