# LeadPages Domain System

LeadPages integrates with Dreamscape/Crazy Domains reseller APIs for domain search, registration, and DNS management.

## Product Goal

Customers should be able to search, buy, and connect domains without leaving LeadPages.

The experience should feel similar to major domain providers while remaining inside the LeadPages interface.

## Important Domain Concepts

- Domain search
- Domain registration
- Domain pricing/margins
- Privacy/private registration where available
- DNS management
- Custom domain connection
- SSL readiness
- Dreamscape account balance/credit
- Payment handling

## Rules

- Do not expose Dreamscape credentials.
- Do not change domain purchase flow without approval.
- Domain registration should remain inside LeadPages.
- Privacy options should be global where relevant, not just Australian domains.
- Any pricing/margin logic must be documented.
