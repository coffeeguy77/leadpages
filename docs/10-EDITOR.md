# LeadPages Editor UX

The editor should feel like the control centre for a business website.

## UX Principles

- Professional, polished, and practical.
- Easy to scan.
- Group related settings together.
- Avoid too many disconnected floating controls.
- Preserve power while reducing visual mess.
- Make advanced settings accessible but not overwhelming.

## Current Concerns To Watch

- Large single-file editor.
- Many settings and pills can become hard to navigate.
- Some account/admin controls may feel disconnected from the main editor.
- Expansion panels should appear near the selected option, not far away at page bottom.

## Suggested Direction

- Use grouped cards.
- Consider 3–4 column layouts where space allows.
- Use local expansion panels beside or beneath selected items.
- Keep sign out/account controls in the main navigation area.
- Keep partner/admin controls organised by role.
