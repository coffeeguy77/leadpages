# LeadPages Design System

LeadPages design should be clean, fast, SaaS-like, and business-focused.

## Style Direction

- Professional dashboard UI
- Strong hierarchy
- Clear grouping
- Polished controls
- Modern but not overdesigned
- Fast-loading interface

## Branding Direction

LeadPages should feel:

- Helpful
- Fast
- Practical
- Smart
- Reliable
- Built for local business

## UI Rules

- Avoid demo-looking placeholder text.
- Avoid cluttered top corners.
- Avoid controls that float away from their section.
- Use clear labels and icons.
- Maintain consistent spacing.
- Make destructive actions obvious.
- Make save/publish/preview states clear.
