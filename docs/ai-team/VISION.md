# Vision — AI Website Team

LeadPages sites should improve through a coordinated specialist team that understands the business, respects real editor capabilities, and never bypasses save/publish controls.

## Principles

1. **Existing websites first** — improve live tenant sites in the normal editor.
2. **Site Brain is the shared memory** — specialists read role-scoped slices, not ad-hoc dumps.
3. **Real capabilities only** — recommendations target Marketplace/editor/renderer-backed features.
4. **Advisory before execution** — Phase 1 proposes; later phases draft through approved tools.
5. **Human approval** — users confirm knowledge and recommendation status; publish stays in the editor.
6. **Fail closed** — deployed storage never silently falls back to process memory.
7. **Website Studio stays On Ice** — generation experiments are not the active product path.

## Success for Phase 1

- A site owner can open **AI Website Team**, confirm Site Knowledge, ask Atlas for a review, and see actionable recommendations with clear status — without any live mutation.
