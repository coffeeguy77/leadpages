# LeadPages SEO System

LeadPages is strongly SEO-focused.

## Important SEO Areas

- Page titles
- Meta descriptions
- Service pages
- Location/suburb pages
- Dynamic sitemap
- Robots configuration
- Schema markup
- Internal links
- Fast page rendering

## Suburb SEO

The app has a Next.js route for suburb SEO pages.

Only suburbs configured in the site's service area should generate pages.

This avoids creating uncontrolled doorway pages.

## Rules

- SEO pages must be useful and locally relevant.
- Do not generate unlimited thin pages.
- Keep suburb pages tied to real service areas.
- Ensure sitemap and robots configuration point to the correct dynamic sitemap.
- AI-written intros should be cached to avoid repeated generation costs.
